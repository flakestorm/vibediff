# VibeDiff Core (OSS)

Rust CLI and semantic analysis engine.

## Scope
- Git diff and intent extraction
- AST/entity mapping
- Local and cloud scoring bridge
- Reporting (CLI/JSON/SARIF)

## First Milestones
1. CLI scaffold (`vibediff check`)
2. TypeScript AST support
3. Ollama local scorer
4. Basic pre-commit integration
