# VibeDiff Frontend Cloud (Next.js)

Next.js application for cloud onboarding, policy management, audit dashboards, and billing.

## Scope
- Authentication and onboarding
- API key setup for CLI integration
- Policy management UI
- Audit history and score dashboards
- Team and billing settings

## First Milestones
1. Next.js scaffold (App Router)
2. Auth/session flow
3. Audit list and detail views
4. Policy editor v1
